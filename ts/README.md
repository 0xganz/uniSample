### ts-uni-sample
    使用命令行交互

### ready
 
 . node v12+
 . cp account.json.example account.json 填写自己的账户信息

### start
```
    cd ts
    npm i
    npm start
```

### sample 

查看 池子信息
```
 1
```

计算 1weth 可兑换 lz  数量

```
 2 WETH 1
```


计算 获取 1weth 需要 输入 lz 数量

```
 3 WETH 1
```

开始兑换 将 0.1 WETH 兑换成 LZ

```
4 WETH 0.1
```

将自己账户的weth token 划转权限授权到 合约
```
5 WETH 
```

查看 tranaction 详情 (开发中...)

```
5 transactionHASH 
```
